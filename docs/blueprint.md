# **App Name**: PersonaPage

## Core Features:

- Persona Selector: Display a landing page that allows users to select which of your personas to view
- Credential Display: Displays professional credentials such as work history and education.
- Project Showcase: Display a curated list of your coding projects with descriptions and links.
- Contact Information: Displays links to contact methods and social media. Links change depending on selected persona
- Dynamic Welcome Message: A tool which uses LLM to write the website's welcome message. Changes depending on the persona selected

## Style Guidelines:

- Primary color: A professional blue (#2E9AFE) for Jamal, a fun orange (#FFB347) for Jamallin, and a playful green (#82E0AA) for Uncle J.
- Background color: For Jamal, a clean, light grey (#F0F0F0); for Jamallin, a vibrant light yellow (#FFFACD); and for Uncle J, a gentle pastel blue (#E0F7FA).
- Accent color: Complementary shades for each persona – a deep navy (#0B0B45) for Jamal, a bright pink (#FF69B4) for Jamallin, and a soft lavender (#E6E6FA) for Uncle J – to highlight interactive elements.
- Professional fonts for Jamal, a friendly, slightly rounded font for Uncle J, and a bold, modern font for Jamallin.
- Use icons relevant to the chosen persona – business icons for Jamal, playful cartoon icons for Uncle J, and modern, abstract icons for Jamallin.
- Clean, structured layout for Jamal; a colorful, whimsical layout for Uncle J; and a dynamic, asymmetrical layout for Jamallin.
- Subtle, professional animations for Jamal; fun, bouncy animations for Uncle J; and energetic transitions for Jamallin.
